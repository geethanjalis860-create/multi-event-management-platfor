Multi Event Management Platform

A full-stack web application that enables users to discover, register, and manage multiple events efficiently. Organizers can create events, manage attendees, and track performance through a dedicated dashboard.

📌 Problem Statement

Managing multiple events is challenging due to scattered tools, manual tracking, and lack of real-time insights. A centralized and scalable solution is needed to simplify event management.

💡 Solution

The platform provides:

Centralized event discovery

Easy event registration

Organizer dashboard with analytics

Secure authentication system

Scalable and user-friendly architecture

🚀 Features
👤 User

Register & Login

Browse and view events

Book/Register for events

View registered events

🧑‍💼 Organizer

Create, edit, delete events

Manage attendees

Track registrations

View analytics dashboard

🛠️ Tech Stack

Frontend: React.js, Tailwind CSS, Axios
Backend: Node.js, Express.js
Database: MongoDB
Deployment: Vercel (Frontend), Render (Backend)

🏗️ Architecture

User → React Frontend → Express Backend → MongoDB

REST APIs

JWT Authentication

Modular backend design
